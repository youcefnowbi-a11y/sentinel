# P4D Browser Supremacy Decision

Date: 2026-04-29
Status: Decision complete

## Decision Options

P4D was required to choose one of:

```text
A) Browser ready for next organ.
B) Browser needs more hardening.
C) Run external benchmark campaign.
```

## Decision

P4D selects:

```text
B + C
```

Browser V3 needs scientific hardening before a final browser claim, and a future
external benchmark campaign is required before any external peer-superiority
claim.

## Why Not A Alone

The browser is strong, but the evidence is still mostly local:

- P4C-S measured nine mission groups locally;
- targeted tests used two iterations per group;
- live adapter harness is fixture-bound;
- Brain/Cortex has not yet deeply mapped every V3 event class;
- CI reporting can understate uncertainty for perfect small-n runs.

Those facts make a plain "ready and finished" decision too broad.

## What Is Locked

These claims are accepted:

```text
Browser V3 authority architecture is coherent.
Browser V3 classes are implemented as governed powers.
ContextPack and ToolIntentCompiler prevent raw LLM execution.
FinalGate covers V3 contracts.
Local fixture/live adapter proof exists.
P4C-S provides useful local regression coverage.
```

## What Is Not Locked

These claims are not accepted:

```text
Browser V3 is scientifically finished.
Browser V3 beats peer browser agents on open web.
Two-run local success proves production stability.
Fixture harness behavior proves broad live-site robustness.
V3 outputs are fully integrated into mission cognition.
```

## Required Next Gate

The next browser gate should be:

```text
P4D-H Browser Scientific Hardening
```

Scope:

1. small-n-safe EvalBench statistics;
2. >=10 local live-adapter runs per mission group;
3. explicit Browser-Cortex mapping for V3 events;
4. adversarial JS no-network corpus;
5. adversarial cookie/storage and HAR/body redaction corpus;
6. forged V3 event negative matrix;
7. self-hosted benchmark plan implementation.

## Supremacy Claim Rule

Sentinel may say:

```text
Browser V3 is governed and locally measured.
```

Sentinel may not yet say:

```text
Browser V3 has externally proven browser supremacy.
```

## Final Verdict

Browser V3 remains one of Sentinel's strongest organs, but P4D does not close
the browser program. It opens the scientific hardening and benchmark campaign
needed to make the final claim real.
