# Product Docs

This folder owns user-facing product specs and UI-oriented product artifacts.

## Files

- `PRODUCT_SPEC.md` - product spec.
- `EVIDENCE_LEDGER_UI_MOCK.md` - evidence ledger UI mock.

## Rule

Product docs describe experience and value. Security, certification, and runtime
authority belong in `mission-os/`, `brain/`, or `audits/`.
