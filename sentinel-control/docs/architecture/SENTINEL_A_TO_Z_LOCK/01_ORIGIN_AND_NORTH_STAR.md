# Origin And North Star

## Origin

Sentinel started from a practical sequence:

```text
Idea
-> evidence
-> decision
-> GTM pack
-> risk review
-> user approval
-> safe execution
-> trace log
```

`RedditPulse/CueIdea` supplies market evidence. `sentinel-control` turns that
evidence into governed mission work. `agent-lab` studies powerful agent systems
without importing vendor runtimes.

## North Star

Sentinel is a Mission OS and Super Agent Foundry:

```text
Mission request
-> Brain L4 cognition
-> MissionAuthorityEnvelope
-> evidence and belief state
-> agent society planning
-> organ selection
-> dry-run / approval / execution when authorized
-> receipts
-> replay
-> FinalGate
```

The final product should act strongly when authority is explicit, but every
action remains bounded, traceable, reversible when possible, and certifiable.

## Innovation Frame

Most visible AI-agent ideas already exist somewhere. Sentinel's novelty is the
organizing principle:

```text
recombine strong agent capabilities
under one authority/proof/control architecture
with persistent mission continuity
and product-grade workflow surfaces
```

This is system convergence, not novelty by isolated component.
