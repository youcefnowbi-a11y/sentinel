# Architecture Docs

This folder owns system-level architecture and roadmap documents.

## Files

- `SENTINEL_AGENT_MASTER_ARCHITECTURE.md` - main mission OS architecture.
- `SENTINEL_AGENT_IMPLEMENTATION_ROADMAP.md` - implementation order.
- `AGENT_FOUNDRY_DIRECTION.md` - foundry direction for future agents/modules.

## Rule

Architecture docs define direction and dependency order. They should not contain
raw vendor dumps, implementation logs, or certification records.
