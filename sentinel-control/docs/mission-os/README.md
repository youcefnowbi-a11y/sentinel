# Mission OS Docs

This folder owns mission-level product rules, security rules, and evidence
contracts.

## Files

- `GTM_OPERATOR_SPEC.md` - GTM operator behavior.
- `CUEIDEA_SENTINEL_EVIDENCE_CONTRACT.md` - CueIdea evidence mapping.
- `FIREWALL_POLICIES.md` - firewall policy baseline.
- `SECURITY_MODEL.md` - product security model.

## Rule

Mission OS docs define user-facing authority and safety behavior. They should
stay aligned with the brain authority docs.
