# Audit And Certification Docs

This folder owns audit reports, implementation reviews, progress reports, and
certification records.

## Files

- `SENTINEL_CORE_V1_CERTIFICATION.md` - current core certification boundary.
- `G12B_IMPLEMENTATION_AUDIT.md` - mission authority implementation audit.

## Rule

Audit docs should separate verified facts, inferred risks, and unresolved
questions. Certification docs must state the exact tested boundary.

Audit docs that name research source systems belong in `agent-lab`, not in this
product docs tree. Final product release artifacts must contain only
Sentinel-native audit and certification records.
